# dotfiles

## Init

```bash
# Install chezmoi and dotfiles from github, empty machine with a single command
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply im-gnar

# already installed chezmoi
chezmoi init im-gnar --apply
```
